# Mini C Compiler

A **Mini C Compiler** implemented using **Lex (Flex)** and **Yacc (Bison)** as part of a **Compiler Design** academic project.
This compiler supports a **subset of the C programming language** and demonstrates all **major phases of compilation**.

---

## Features

* Lexical Analysis using **Lex / Flex**
* Syntax Analysis using **Yacc / Bison**
* Symbol Table management
* Semantic Analysis (undeclared & redeclared variables)
* Arithmetic expressions: `+  -  *  /  =`
* Three Address Code (TAC) generation
* Assembly-like Target Code generation
* Easy-to-understand and viva-friendly design

## How to Build & Run

### Generate Lexer & Parser

```bash
win_flex lex.l
win_bison -d parser.y
```

### Compile the Compiler

```bash
gcc lex.yy.c parser.tab.c -o minic
```

### Run the Compiler

```bash
minic < test.c
```

## Limitations

* Supports only a **subset of C language**
* No control flow statements (`if`, `while`, `for`)
* No function calls except `main`
* No code optimization techniques
